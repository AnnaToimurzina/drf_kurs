Habit Tracker
Habit Tracker - это веб-приложение для отслеживания полезных привычек, разработанное в рамках курсового проекта. Приложение позволяет пользователям создавать, отслеживать и улучшать свои привычки.

Функциональность
Регистрация и авторизация: Пользователи могут создавать учетные записи и входить в систему.
Cоздание привычек: Пользователи могут добавлять новые привычки, указывая действие, место, время и другие параметры.
Редактирование и удаление привычек: Пользователи могут изменять параметры существующих привычек или удалять их.
Просмотр списка привычек: Пользователи могут просматривать список своих привычек, а также публичные привычки других пользователей.
Получение напоминаний: Система отправляет уведомления пользователю по электронной почте и через Telegram для напоминания о выполнении привычек.

Интеграция с Telegram
Для настройки напоминаний через Telegram в проекте используется Celery. Реализована интеграция с мессенджером, чтобы отправлять уведомления в указанное время.

## Запуск проекта

1. Склонируйте репозиторий:

    git clone https://github.com/AnnaToimurzina/drf_kurs.git
    cd course


2. Запустите проект с помощью Docker Compose:
docker-compose build
docker-compose up

3. После успешного запуска, ваше Django-приложение будет доступно по адресу [http://localhost:8000/](http://localhost:8000/).

4. Чтобы остановить проект, выполните команду: docker stop config


**Примечание:**
- Убедитесь, что у вас установлен Docker и Docker Compose перед запуском проекта.
